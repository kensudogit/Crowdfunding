## サンプルデータの生成方法

### 方法1: 管理画面から生成（推奨）

1. アプリケーションにログイン
2. ヘッダーの「管理画面」をクリック
3. 「サンプルデータを生成」ボタンをクリック
4. 生成完了メッセージを確認

### 方法2: APIから直接生成

```bash
curl -X POST http://localhost:8000/api/sample-data/generate
```

### 方法3: データベース初期化時に自動生成

Dockerコンテナを再作成すると、`database/init/02_sample_data.sql`が自動的に実行されます。

```bash
docker-compose down -v
docker-compose up -d
```

- **本番環境では使用しないでください**: この機能は開発・プレゼン用です
